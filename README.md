# Sales Performance Dashboard | Power BI

Overview

This project presents an interactive Power BI dashboard developed using the SampleSuperstore.csv dataset from Kaggle.

The objective was to transform raw sales data into meaningful business insights through data cleaning, modeling, DAX measures, and interactive visualizations. To analyze and visualize sales performance dashboard based on -  1. Region, Category, Segment 2. Profitability 3. Discounts Impact 4. Shipping Trends
